Grid Item

grid-area

Define a área do item do grid. É um atalho para grid-row-start, grid-column-start, grid-row-end, grid-column-end.

        grid-template-columns: repeat(3, 1fr);
        grid-auto-rows: 50px;

        ----------------------------------------------

        .grid-area-1 .item-1 {
          grid-area: 1 / 2 / 4 / 3;
          grid-row-start: 1;
          grid-column-start: 2;
          grid-row-end: 4;
          grid-column-end: 3;
        }

        .grid-area-1 .item-5 {
          grid-area: span 2 / 3;
          grid-row-start: span 2;
          grid-column-start: 3;
        }

item-1 - grid-area: 1 / 2 / 4 / 3
grid-row-start: 1;
grid-column-start: 2;
grid-row-end: 4;
grid-column-end: 3;
ou seja: coluna 2, linhas 1 a 3

item-5:
coluna 3, linhas 2 e 3
