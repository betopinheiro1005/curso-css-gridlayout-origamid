Grid Container

justify-content

Justifica os itens do grid em relação ao eixo x (horizontal).

        grid-template-columns: repeat(3, 100px);
        grid-template-rows: 100px 100px 100px;

        .item-2, .item-3 {
          grid-row: span 2;
        }
        .item-5 {
          grid-column: span 2;
        }
