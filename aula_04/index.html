Grid Container

grid-template-areas

Define áreas específicas no grid. O ponto (.) pode ser utilizado para criar áreas vazias.

      grid-template-areas:
        "logo     nav     nav"
        "sidenav  content advert"
        "sidenav  footer  footer";
