Grid Container

grid-gap

Define o gap (gutter) entre os elementos do grid.

        grid-template:
          "logo     nav     nav"    50px
          "sidenav  content advert" 150px
          "sidenav  footer  footer" 100px
          / 100px 1fr 50px;

          gap: 20px;

Espaçamento entre linhas e colunas
Com gap o espaçamento é apenas interno

        column-gap: 20px;
